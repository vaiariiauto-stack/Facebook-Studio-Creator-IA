# CRM IA avec Notion et Make

## Objectif
Créer un CRM dans Notion connecté à Make pour automatiser
la gestion des prospects.

## Outils
- Notion
- Make
- Claude
- ChatGPT

## Fonctionnement

1. Un prospect remplit un formulaire.
2. Les données sont envoyées dans Notion.
3. Make déclenche un scénario.
4. L'IA qualifie automatiquement le prospect.
5. Une réponse personnalisée est générée.
